# DesktopOverlay

DesktopOverlay is a Windows app that lets you place an image on top of your desktop with adjustable transparency.

## Download And Install

1. Open the latest GitHub Release.
2. Download `DesktopOverlaySetup.exe`.
3. Run `DesktopOverlaySetup.exe`.
4. Launch DesktopOverlay from the Start Menu.

## Uninstall

1. Open Windows Settings.
2. Go to Apps > Installed apps.
3. Find DesktopOverlay.
4. Click Uninstall.

## Features

- Load an image from disk
- Show a preview and file info in the control panel
- Drag the overlay to move it
- Drag the bottom-right corner to resize it
- Adjust overlay opacity
- Lock and unlock click-through mode

## How To Use

1. Open DesktopOverlay.
2. Click Open Image.
3. Position and resize the overlay on screen.
4. Use Lock Overlay when you want to click through to other apps.
5. Use Unlock Overlay when you want to move or resize again.

## Tech Notes

- Built with PySide6
- Packaged with PyInstaller
- Installer built with Inno Setup
